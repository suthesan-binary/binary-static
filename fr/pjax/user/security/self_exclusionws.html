Auto-Exclusion

Le trading en ligne peut être addictif. L'auto-exclusion est un moyen de limiter votre activité de trading en ligne, si vous en avez besoin. Vous pouvez fixer des limites dans votre compte pour aider à prévenir les pertes non désirées. Vous pouvez ajuster ces limites à tout moment, mais ne les supprimer qu'après 24 heures. Contactez-nous pour supprimer ou ajuster vos limites.

Vous pouvez également nous demander de vous exclure du trading sur Binary.com pour une période de temps spécifique. Une fois fixée, cette date d'auto-exclusion ne peut être modifiée.

Si vous envisagez de vous auto-exclure, vous pouvez vous inscrire auprès de GAMSTOP.

GAMSTOP est un service gratuit qui vous permet de vous auto exclure de toutes les sociétés de jeu en ligne sous licence en Grande-Bretagne.

Pour trouver plus d'information et pour signer avec GAMSTOP, veuillez visiter www.gamstop.co.uk.
